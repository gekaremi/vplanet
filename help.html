** Note that long help is not fully implemented.** Command Line Options ~~~~~~~~~~~~~~~~~~~~

-v, -verbose

Maximum verbosity, i.e. display all warnings and updates.

-q, -quiet

No verbosity, i.e. nothing printed to device.

-h, -help

Display the short help message.

-H, -Help

Display the extended help message.

Input File

Comments/White Space: Any characters to the right of a # sign are treated as a comment and are ignored. All white space is ignored. Options that take an array may span multiple lines if a $ is placed at the end of the line. Options must be the first string on any line, and must be written exactly as listed below. Arguments may have any format, and need only be unambiguous.
